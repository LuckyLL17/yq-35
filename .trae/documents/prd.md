## 1. 产品概述

人类反应测试网站——一个多维度认知与反应能力测评平台，涵盖反应速度、记忆、打字、瞄准、视觉辨识等 10+ 种测试项目。用户可通过游戏化测试了解自身各项能力水平，并与全球平均值对比。目标用户为对所有年龄段对自身能力好奇的用户，核心价值在于趣味性与科学性的结合。

## 2. 核心功能

### 2.1 用户角色

| 角色 | 注册方式 | 核心权限 |
|------|----------|----------|
| 访客 | 无需注册 | 浏览首页、参与所有测试、查看结果 |
| 注册用户 | 邮箱注册 | 保存历史成绩、能力雷达图、全球排名 |

### 2.2 功能模块

1. **首页**: 英雄区、测试项目卡片网格、能力总览入口
2. **测试页**: 10+ 种独立测试，每种测试拥有独立的交互界面和计分系统
3. **结果页**: 单项成绩展示、全球百分位对比、能力雷达图

### 2.3 测试项目清单

| 序号 | 测试名称 | 类型 | 描述 |
|------|----------|------|------|
| 1 | 反应时间 | 反应 | 屏幕变色后尽快点击，测量反应速度（毫秒） |
| 2 | 序列记忆 | 记忆 | 记忆逐渐增长的数字序列，类似 Simon 游戏 |
| 3 | 打字速度 | 打字 | 在限定时间内输入给定文本，计算 WPM 和准确率 |
| 4 | 瞄准测试 | 瞄准 | 点击随机出现的目标圆点，测量精准度和速度 |
| 5 | 黑猩猩测试 | 记忆 | 短暂显示数字后消失，按顺序点击数字位置 |
| 6 | 视觉记忆 | 视觉 | 记忆网格中亮起的方块位置，难度递增 |
| 7 | 言语记忆 | 记忆 | 记忆并回忆显示的单词序列 |
| 8 | 颜色匹配 | 视觉 | 判断文字颜色与文字内容是否一致（Stroop 效应） |
| 9 | 时间感知 | 感知 | 估计给定时间间隔的长度，测试内在时钟 |
| 10 | 手速测试 | 反应 | 在限定时间内尽可能快地点击按钮，计算 CPS |
| 11 | 数字速算 | 计算 | 快速心算数学题，测试计算速度和准确率 |
| 12 | 节奏感知 | 感知 | 感知并复现节奏模式 |

### 2.4 页面详情

| 页面名称 | 模块名称 | 功能描述 |
|----------|----------|----------|
| 首页 | 英雄区 | 大标题、副标题、动态背景粒子动画、CTA 按钮 |
| 首页 | 测试卡片网格 | 12 个测试项目卡片，含图标、名称、简介、难度标签 |
| 首页 | 能力总览入口 | 引导用户完成全部测试后生成雷达图 |
| 测试页 | 测试区域 | 各测试的核心交互区域，含倒计时、计分、状态提示 |
| 测试页 | 操作指引 | 测试前展示规则说明和操作演示 |
| 结果页 | 成绩展示 | 本次得分、用时、准确率等核心指标 |
| 结果页 | 百分位对比 | 与全球平均值的对比条形图 |
| 结果页 | 能力雷达图 | 综合展示各项能力维度 |
| 结果页 | 操作按钮 | 重新测试、分享结果、返回首页 |

## 3. 核心流程

用户进入首页 → 选择测试项目 → 阅读测试规则 → 开始测试 → 完成测试 → 查看结果 → 与全球数据对比 → 选择重新测试或尝试其他测试

```mermaid
flowchart TD
    "首页" --> "选择测试项目"
    "选择测试项目" --> "阅读规则"
    "阅读规则" --> "开始测试"
    "开始测试" --> "测试交互"
    "测试交互" --> "测试结束"
    "测试结束" --> "查看结果"
    "查看结果" --> "全球对比"
    "全球对比" --> "重新测试"
    "全球对比" --> "返回首页"
    "返回首页" --> "选择测试项目"
```

## 4. 用户界面设计

### 4.1 设计风格

- **主色调**: 深色背景 (#0a0a0f) + 霓虹绿 (#00ff88) 作为强调色 + 暗紫 (#1a1a2e) 作为次级色
- **次级色**: 电子蓝 (#00d4ff)、警示红 (#ff3366)、温暖橙 (#ff8800)
- **按钮风格**: 圆角胶囊形，霓虹发光效果，hover 时发光增强
- **字体**: 标题使用 Orbitron（科技感），正文使用 Rajdhani（清晰现代）
- **布局风格**: 卡片式网格布局，顶部导航栏，深色赛博朋克风格
- **图标风格**: 线性图标 + 霓虹发光，赛博朋克风格
- **动画**: 粒子背景、卡片 hover 发光、测试中脉冲动画、成绩揭晓动画

### 4.2 页面设计概述

| 页面名称 | 模块名称 | UI 元素 |
|----------|----------|---------|
| 首页 | 英雄区 | 大号 Orbitron 标题、霓虹绿渐变文字、粒子背景、脉冲 CTA 按钮 |
| 首页 | 测试卡片网格 | 3x4 网格、深色卡片 + 边框发光、图标 + 名称 + 简介布局、hover 上升+发光效果 |
| 测试页 | 测试区域 | 居中大面板、顶部状态栏（时间/分数/进度）、底部操作区 |
| 测试页 | 操作指引 | 半透明覆盖层、步骤图示、倒计时开始 |
| 结果页 | 成绩展示 | 大号数字动画递增、环形进度条、发光边框 |
| 结果页 | 百分位对比 | 水平条形图、用户值 vs 全球平均值、颜色编码 |
| 结果页 | 能力雷达图 | Canvas 绘制雷达图、霓虹绿填充、各维度标签 |

### 4.3 响应式设计

- 桌面优先设计，最小宽度 1024px 优化
- 平板端（768px-1024px）卡片 2 列布局
- 移动端（<768px）单列卡片，测试区域自适应缩放
- 触摸优化：点击区域足够大，防止误触

### 4.4 动效设计

- 页面加载：卡片依次淡入 + 上升
- 测试进行中：心跳脉冲边框动画
- 点击反馈：涟漪扩散 + 缩放弹性
- 成绩揭晓：数字从 0 快速递增到最终值
- 雷达图：从中心向外展开绘制
