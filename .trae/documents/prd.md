# 人类反应测试网站 - 产品需求文档 (PRD)

## 1. 项目概述
### 1.1 项目名称
Human Benchmark - 人类反应测试平台

### 1.2 项目简介
一个多功能的人类能力测试网站，提供反应时间、记忆力、打字速度、瞄准能力、视觉辨别等多维度的认知和生理能力测试。借鉴经典的 Human Benchmark 网站，打造一个现代化、视觉精美的测试平台。

### 1.3 目标用户
- 想测试自己各项能力的普通用户
- 游戏玩家测试反应速度和瞄准能力
- 学生测试记忆力和认知能力
- 对人类极限感兴趣的科技爱好者

### 1.4 设计风格
**暗黑科技风格 (Dark Tech)**：深色调 + 霓虹色彩点缀 + 玻璃拟态效果 + 流畅动画，营造出未来感和科技感。

---

## 2. 功能需求

### 2.1 核心功能模块

| 测试名称 | 测试描述 | 核心玩法 |
|---------|---------|---------|
| 反应时间 | 测试视觉反应速度 | 等待屏幕变绿后尽快点击 |
| 数字记忆 | 测试短时数字记忆能力 | 短暂展示一串数字，用户回忆输入 |
| 打字速度 | 测试打字速度和准确率 | 在限定时间内输入给定文本 |
| 瞄准测试 | 测试手眼协调和瞄准能力 | 尽快点击随机出现的目标 |
| 黑猩猩测试 | 测试工作记忆能力（经典） | 记住数字位置，按顺序点击 |
| 颜色视觉 | 测试颜色辨别能力 | 找出颜色不同的方块 |
| 序列记忆 | 测试视觉序列记忆（Simon Says） | 记住并重复灯光闪烁序列 |
| Stroop 效应 | 测试认知干扰控制能力 | 说出文字的颜色而非文字含义 |
| 数学速度 | 测试心算速度和准确性 | 快速计算简单数学题 |

### 2.2 通用功能
- **成绩记录**：每次测试后展示成绩和统计数据
- **历史最佳**：使用 localStorage 保存各测试的历史最佳成绩
- **重试功能**：每次测试后可立即重试
- **返回首页**：随时返回测试选择界面
- **响应式设计**：支持桌面端和移动端

### 2.3 首页
- 展示所有测试的卡片式列表
- 每个卡片显示：图标、名称、简短描述、历史最佳成绩
- 悬停动效

---

## 3. 非功能需求

### 3.1 性能要求
- 首屏加载时间 < 2s
- 动画帧率保持 60fps
- 测试计时精度达到毫秒级

### 3.2 兼容性要求
- 支持主流浏览器（Chrome、Safari、Firefox、Edge）
- 响应式适配移动端

### 3.3 数据持久化
- 使用 localStorage 保存历史最佳成绩
- 无需后端服务，纯前端实现

---

## 4. 用户流程

1. 用户进入首页 → 浏览所有测试卡片
2. 点击某个测试卡片 → 进入测试页面
3. 阅读测试说明 → 开始测试
4. 完成测试 → 查看成绩和历史最佳
5. 选择重试或返回首页

---

## 5. 视觉与交互规范

### 5.1 配色方案（暗黑科技主题）
- 主背景：深灰黑色 `#0a0a0f`
- 次背景：稍浅的深色 `#12121a`
- 卡片背景：半透明玻璃拟态
- 主色：青蓝色 `#00d4ff`
- 强调色：紫色 `#a855f7`、粉色 `#ec4899`、绿色 `#10b981`、红色 `#ef4444`
- 文字：白色 `#ffffff` / 灰色 `#9ca3af`

### 5.2 字体
- 标题：使用现代感的无衬线字体
- 正文：清晰易读的无衬线字体
- 数字/计时：等宽字体

### 5.3 动画
- 页面切换过渡动画
- 卡片悬停发光效果
- 按钮点击反馈
- 测试结果展示的渐入动画
- 计时器的脉冲效果
